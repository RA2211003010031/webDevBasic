CSS BASICS EXPLAINED

So p is the selector and we write the syntax like

p {color : red;}

 here, color is the property
             red is the value
Here, selector is basically the tag name or id name which you made in html code and property is what the property of element you want to change and at last the value is what value to place there.

So we can add multiple selectors using comma as seperator

header, p.intro


So we write further the action block or declaration block as-

header, p.intro
{
background-color : red
border-radius : 3px
)


So there are three different ways to include css to markup. Markup is nothing but html code.
1. INLINE CSS In this, the css is directly added to the markup code. Like we were doing for the img, that's where the width and length i guess.
2. INTERNAL CSS In this, the css is kept inside the head. The css code is written there without disturbing the body code of markup.
3. EXTERNAL CSS In this, the css is kept inside a .css file and is included in the markup. This is the best method to maintain the skeleton and style of website.
